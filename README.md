# Apple Global Sales Analysis

## Project Overview
This project analyzes Apple product sales across different countries, product categories, and sales channels.  
The analysis focuses on revenue generation, customer behavior, and product performance using SQL queries and a Tableau dashboard.

---

## Project Objectives
- Analyze total revenue generated from global sales
- Identify top-performing products and categories
- Evaluate sales trends over time
- Understand customer segments and payment methods
- Visualize insights through an interactive dashboard

---

## Tools & Technologies Used

- Excel (Data Cleaning)
- SQL (Data Analysis)
- Tableau (Data Visualization)
- GitHub (Project Documentation)

---

## Dataset Features

The dataset contains the following fields:

- Sale ID
- Sale Date
- Year / Quarter / Month
- Country / Region / City
- Product Name
- Product Category
- Storage Capacity
- Product Color
- Unit Price
- Discount Percentage
- Units Sold
- Revenue
- Sales Channel
- Payment Method
- Customer Segment
- Customer Rating
- Return Status

---

## Data Cleaning Steps

The dataset was cleaned using Excel before analysis:

- Removed duplicate records
- Standardized column names
- Checked missing values
- Formatted date columns
- Verified numerical columns

---

## SQL Analysis

Key SQL queries performed:

1. Total revenue generated from all sales
2. Total units sold across all products
3. Distinct product categories
4. Average unit price of Apple products
5. Top 5 cities with highest revenue
6. Revenue by product category
7. Revenue by country
8. Most used payment method
9. Monthly sales trend
10. Top 5 revenue generating products

---

## Dashboard

An interactive Tableau dashboard was created to visualize the insights.

Dashboard includes:

- Total Revenue KPI
- Units Sold KPI
- Average Customer Rating
- Revenue by Product Category
- Monthly Sales Trend
- Revenue by Country Map
- Sales Channel Performance
- Payment Method Distribution

---

## Key Insights

- Mac products generated the highest revenue
- iPhone category showed consistent global demand
- Online sales channel contributed major revenue
- Some countries generated significantly higher sales
- Customer ratings remained consistently high across products
